# Mexicans Catering Website

A vibrant, animated website for "Mexicans" - an authentic Mexican catering business.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Eye-catching animations and transitions throughout
- **Bold Mexican Aesthetic**: Authentic colors and design elements
- **Menu Showcase**: Display of catering specialties with pricing
- **Service Information**: Full-service catering, delivery, and live cooking stations
- **Contact Section**: Easy-to-find contact information

## Live Demo

Visit the live site at: `https://yourusername.github.io/repository-name/`

## Setup

1. Clone or download this repository
2. Open `index.html` in any web browser
3. No build process or dependencies required - it's a single HTML file!

## GitHub Pages Deployment

1. Upload `index.html` to your GitHub repository
2. Go to Settings ‚Üí Pages
3. Select your branch (usually `main`)
4. Click Save
5. Your site will be live in a few minutes!

## Customization

To customize the website:
- Edit contact information in the contact section
- Modify menu items and pricing
- Change colors in the CSS `:root` variables
- Add your own images or content

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Google Fonts (Bebas Neue, Comfortaa, Libre Baskerville)
